Policy Administration System (PAS4 Standard)

This project involves developing a comprehensive policy administration system that adheres to the PAS4 standards, utilizing the lean backend frameworks. The system aims to streamline the management of insurance policies, including policy creation, updates, and compliance monitoring.

Features

  • Policy Creation and Management
  • Compliance Monitoring
  • Automated Notifications
  • Reporting and Analytics

Technologies Used

  • Django
  • R
  • MySQL
  • SQL
