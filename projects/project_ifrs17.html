IFRS 17 Reporting System

This project focuses on creating a reporting system to generate IFRS 17 compliant financial statements. The goal is to facilitate accurate and efficient financial reporting for insurance companies, ensuring compliance with the latest accounting standards.

Worked on a custom Excel Add-in for users to enter data that would be stored in a cloud service and later used by the code to create reports dased on the data stored.

Features

  • Automated Report Generation
  • Compliance with IFRS 17 Standards
  • Data Integration and Analysis
  • Customizable Report Templates

Technologies Used

  • R
  • Office JS
  • Microsoft Visual Basic for Applications
  • Microsoft Excel
  • Google Cloud Run
