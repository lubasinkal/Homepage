IFRS 17 Reporting System

This project focuses on creating a reporting system using R to generate IFRS 17 compliant financial statements. The goal is to facilitate accurate and efficient financial reporting for insurance companies, ensuring compliance with the latest accounting standards.

Features

  • Automated Report Generation
  • Compliance with IFRS 17 Standards
  • Data Integration and Analysis
  • Customizable Report Templates

Technologies Used

  • R
  • Shiny
  • Microsoft Visual Basic for Applications
  • Microsoft Excel
